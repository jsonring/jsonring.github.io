项目概述

ELK Community（易聊社区）的核心定位是：国内首款主打「企业级全场景」的 Java 开源社区门户，支持内部协作 + 外部品牌门户一体化，适配金融、电商、制造等中大型企业的高并发需求。

技术栈

Spring Cloud Spring Boot 2.x MyBatis-Plus Redis Elasticsearch Kafka Vue3 Element Plus

采用微服务架构，支持分布式部署。

核心企业级功能

1. 权限体系

  • RBAC 3.0（角色 + 数据权限）
  • 多租户隔离（集团/子公司数据隔离）
  • SSO 单点登录（对接 CAS/OAuth2/企业微信/钉钉/AD域/LDAP）

2. 门户能力

  • 拖拽式自定义门户模板（品牌展示、企业资讯、社区互动、产品反馈一体化）
  • 多端适配（PC/APP/小程序/H5）

3. 企业协作

  • 内置知识库
  • 问答工单
  • 内部话题讨论
  • 文件共享（对接 MinIO/阿里云 OSS）
  • 跨部门协作标签

4. 安全合规

  • 操作审计日志
  • 敏感词过滤
  • 数据脱敏
  • 内容多级审批流程
  • 防爬虫/防灌水
  • 等保三级适配

5. 运营管控

  • 企业级数据看板（用户活跃度、内容转化率、部门协作效率）
  • 用户画像
  • 企业福利积分体系

6. 高可用

  • 读写分离
  • 缓存击穿防护
  • ES 亿级数据全文检索
  • 异地多活适配

项目优势

  • 纯企业级设计
  • 中文文档完善
  • 维护活跃（月均更新）
  • 国内大量企业落地案例

适配场景

  • 中大型企业外部品牌社区
  • 内部 + 外部一体化门户
  • 高并发行业社区（金融/电商）
