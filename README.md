# JsonRing GitHub Pages

This repository hosts the GitHub Pages site for jsonring.github.io.

## Contents

- **index.html** - Main landing page with links to project documentation
- **elk-community.html** - Documentation page for ELK Community (易聊社区) project
- **html/** - Gaming blog website template
- **resume/** - Resume templates
- **tooth-shop/** - Dental clinic website template
- Various other project folders

The ELK Community (易聊社区) documentation page provides information about an enterprise-level Java open source community portal project.
